h1. Facebook Graph API OAuth Example

A simple Sinatra-based example of the new Facebook Graph API.

h2. Setup

All you need to do is to configure your OAuth identity:

<code><pre>@client_id = "YOUR-CLIENT-ID-HERE"
@client_secret = "YOUR-CLIENT-SECRET-HERE"</code></pre>

That's it? Yep!

Now you can give it a spin:

<code><pre>ruby facebook-oauth-example.rb</pre></code>
